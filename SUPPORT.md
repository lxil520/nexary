# Support

## 中文

当前仓库已公开，但 GitHub Discussions 尚未开启。问题、使用反馈和需求建议请先使用 GitHub Issues。

提交问题时请包含：

- Java 版本
- Spring Boot 版本
- 使用的 Nexary 模块
- 期望行为与实际行为
- 最小可复现示例或可运行命令

安全漏洞不要发公开 Issue，请按 [SECURITY.md](SECURITY.md) 处理。

## English

The repository is public, but GitHub Discussions are not enabled yet. Use GitHub Issues for questions, usage feedback, and feature requests.

When opening an issue, include:

- Java version
- Spring Boot version
- Nexary module
- expected behavior and actual behavior
- a minimal reproduction or runnable command

Do not report exploitable security issues in public issues. Follow [SECURITY.md](SECURITY.md) instead.
